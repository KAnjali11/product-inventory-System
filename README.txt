SETUP GUIDE

1. Install Node.js

2. Open terminal inside backend folder

3. Run:
npm install

4. Start server:
node server.js

5. Open frontend/index.html in browser

Server URL:
http://localhost:5000

MongoDB:
This project uses local MongoDB:
mongodb://127.0.0.1:27017/inventoryDB

If you want MongoDB Atlas, replace the connection string in server.js.
